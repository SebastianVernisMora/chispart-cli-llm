Acerca de Chispart Mobile

Versión: 1.0.0

Descripción: Terminal de IA Universal optimizado para dispositivos móviles y Termux.

Chispart Mobile es una aplicación PWA (Progressive Web App) que te permite interactuar con múltiples APIs de IA desde tu dispositivo móvil.

Características Principales

  • 🤖 Múltiples APIs: Blackbox AI, OpenAI, Anthropic, etc.
  • 📱 Optimizado para Móviles y Termux
  • 🔒 Seguridad Avanzada Opcional para API Keys
  • 📴 Modo Offline con PWA
  • 🎨 Interfaz Adaptativa con temas
